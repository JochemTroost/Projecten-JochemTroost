# Eigen game project P2

Uitleg over de game:
* de game is een beetje zoals de oude mario, waar je zelf in 2d links en rechts kan bewegen en kan springen.
* Je kan in de game met een pistool je vijanden doodschieten.
* Je gaat in de game door deuren heen aan de rechterkant van het level. 
* Door vijanden dood te schieten krijg je punten, heb je na het laatste level een bepaalde hoeveelheid punten win je.

Uitleg over de besturing:
* Je bestuurd je karakter met de knoppen A en D. Je kan springen met spatie en schieten met S.
* Je kan met de escape knop naar het pauze menu gaan, en hier kan je opnieuw beginnen of stoppen.
