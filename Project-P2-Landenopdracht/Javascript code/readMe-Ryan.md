# Project P2 Game

#

## Wat doe je in de game?
* De game is een beetje zoals space invaders, waar je vijanden moet afschieten die op je af komen en zoveel mogelijk punten moet verdienen.
* Je moet zo veel mogelijk punten verzamelen en jezelf uitdagen jezelf te verbeteren

#

## Hoe werkt de game?
* Je bestuurt jou blokje (het blauwe blokje) met WASD en spatie, waar de knoppen allemaal een functie hebben uiteraard.
* W = naar boven, S = naar beneden, A = naar links, D = naar rechts en spatie is schieten.
* De vijanden komen met willekeurige bewegingen op je af en je kan de vijanden ontwijken, al krijg je hier geen punten voor.
* Je moet zoveel mogelijk vijanden afschieten voor punten
* Je kan ook het spel nog op pauze zetten met P.
