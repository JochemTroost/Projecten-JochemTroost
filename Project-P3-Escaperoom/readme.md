# Project P3 Escaperoom

#

## Algemene informatie

- Thema: Gespecialiseerde inrichting 
- Doel: Weten wat er met Roduriqe Freidrich uit de inrichting is gebeurt 
#

## Inleiding

In deze Escaperoom is het de bedoeling dat de speler een onderzoek gaan uitvoeren. ze zoeken uit wat er met Roduriqe Freidrich is gebeurt. tijdens dat onderzoek gebeuren er merkwaardige dingen. 

## Indeling
In de inrichting zijn verschillende ruimtes die de spelers gaan betreden.
- Ruimte 1: Lab 
- Ruimte 2: Inrichting
- Ruimte 3: cellencomplex
- Ruimte 4: mortuarium
